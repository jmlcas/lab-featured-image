== Description ==

In the settings page check or uncheck the required options: Require Featured Image in Posts and Add Featured Image to Admin Post Columns. Hit "Save Changes"  
If fail to add a featured image to a post before publishing it they'll simply find it impossible to publish.
This plugin adds a column to any post type which supports a featured image.
See which posts have a featured image at a glance.

== Installation ==

1. Upload the entire `lab-featured-image` folder to your `/wp-content/plugins` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Go to Settings>Lab Featured Image - In the settings page check or uncheck the required options: Require Featured Image in Posts and Add Featured Image to Admin Post Columns. Hit "Save Changes" -
